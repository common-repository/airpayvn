== Description ==
Tích hợp cổng thanh toán Airpay.vn vào Woocommerce.

== Installation ==
1. Upload thư mục 'airpayvn-woocommerce-wordpress' vào wp-content/plugins
1. Kích hoạt plugin trong khu vực Plugins -> Installed Plugins
1. Vào Woocommerce -> Settings -> Payment Gateways và chọn Airpayvn
